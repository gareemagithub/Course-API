A simple RESTful API for managing courses using Java and Spring Boot. This API supports basic CRUD operations for adding, retrieving, updating, and deleting courses.
It Contains course id, course name, course price
Features
    Add a New Course
    Get All Courses
    Get a Course by ID
    Update a Course
    Delete a Course
Technology Stack
    Java 17
    Spring Boot
    Spring Data JPA
    MySQL
